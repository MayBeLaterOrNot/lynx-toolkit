Lynx Toolkit Document model

Introduction

Document model

The document model is influenced by HTML and FlowDocument.

Wiki syntax

The "OWiki" syntax was designed to be easy and efficient to parse. It was influenced by Markdown, Creole and Confluence, but the syntax is stricter.

Output formats

The document model can be formatted to the following formats

  • OWiki
  • Markdown wiki
  • Creole wiki
  • Confluence wiki
  • Codeplex wiki
  • XHtml
  • FlowDocument
  • OpenXML (Word)

Examples

Headings

Headings are prefixed with = characters as in Creole. Note that the headings should not be suffixed by =.The markdown syntax was not chosen because it is more difficult to implement in the parser.
